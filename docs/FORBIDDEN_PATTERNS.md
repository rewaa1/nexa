# Forbidden Patterns

Never generate:

- Generic agency websites
- Generic SaaS landing pages
- Generic hero sections
- Generic floating cards
- Generic feature grids
- Generic testimonials sections
- Generic pricing sections
- Generic startup copy
- Generic gradient blobs
- Generic Framer templates

---

Avoid:

"We build digital solutions"

"Transform your business"

"Innovative experiences"

"Trusted partner"

"Cutting-edge technology"

These phrases are banned.

---

Orbix must never resemble a template.
