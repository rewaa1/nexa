# Motion System

## Philosophy

Motion is communication.

Nothing moves without purpose.

---

## Forbidden

- Fade in
- Fade out
- Basic slide up
- Generic Framer animations

---

## Preferred

- Orbital movement
- Gravitational attraction
- Camera travel
- Spatial transitions
- Morphing geometry
- Dynamic systems
- Procedural animation

---

## Scroll Behavior

Scrolling should feel like:

Moving through space.

Not moving through a webpage.

---

## Interaction Behavior

Cursor interactions should influence nearby objects.

Objects should react as if they exist within a physical system.
